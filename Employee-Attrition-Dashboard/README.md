# Employee Attrition & Workforce Insights Dashboard

## Project Overview
This Power BI dashboard analyzes employee attrition trends, workforce demographics, overtime impact, and employee risk segmentation to identify key retention challenges.

## Key Insights
- Higher attrition observed among employees aged 26–35
- Overtime employees show increased turnover risk
- Early-tenure employees contribute significantly to attrition
- Research & Development department shows highest attrition volume

## Tools Used
- Power BI
- DAX
- Data Visualization
- HR Analytics

## Dashboard Features
- Executive KPI Overview
- Workforce Risk Segmentation
- Attrition Analysis by Age, Gender, and Tenure
- Job Satisfaction vs Attrition
- Department-wise Attrition Insights
