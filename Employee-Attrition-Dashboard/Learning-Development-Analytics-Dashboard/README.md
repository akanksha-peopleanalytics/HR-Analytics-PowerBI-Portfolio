# Learning & Development Effectiveness Analytics Dashboard

## Project Overview
This Power BI dashboard analyzes learning participation, workforce capability, training effectiveness, employee engagement, and workforce performance trends.

## Key Insights
- Production department shows highest learning participation
- Training completion rate stands at 50.8%
- High-performing employees demonstrate stronger engagement levels
- Workforce risk is concentrated among lower-performing groups

## Tools Used
- Power BI
- DAX
- HR Analytics
- Workforce Capability Analysis

## Dashboard Features
- Learning Participation Analysis
- Training Outcome Distribution
- Workforce Engagement Analysis
- Workforce Risk Segmentation
- Performance Group Analysis
- Capability & Learning Impact Insights
